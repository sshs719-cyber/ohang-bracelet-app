# 오행운 팔찌 사주 리포트

생년월일, 태어난 시각, 이름만 입력하면 무료 오행 성향 리포트를 보여주고, 부족한 기운에 맞는 오행 팔찌 구매 CTA로 연결하는 정적 웹앱입니다.

## 주요 기능

- 로그인 없는 무료 사주 오행 분석 입력
- 한글 이름 소리 기반 오행 보정
- 목·화·토·금·수 부족 기운별 풍수 추천
- 부족한 기운에 맞는 오행 팔찌 추천 및 구매 CTA
- GitHub Pages, Netlify, Vercel 무료 배포 가능

## 파일 구성

- `index.html`: 홈페이지, 입력 폼, 결과 화면
- `styles.css`: 반응형 디자인
- `app.js`: 오행 계산과 추천 로직
- `assets/product-bracelets.png`: 메인 제품 이미지

## 구매 링크 교체

`app.js`의 `buyButton.href`를 실제 스마트스토어, 카페24, 톡채널, 결제 링크로 바꾸면 됩니다.

```js
buyButton.href = "https://smartstore.naver.com/스토어주소/products/상품번호";
```

## 안내

이 앱의 무료 분석은 자기이해와 웰니스 목적의 콘텐츠입니다. 의료, 법률, 투자 판단을 대체하지 않습니다.
